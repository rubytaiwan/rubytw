layout: post
---
睽違三年，Ruby Taiwan 社群實體活動「Ruby Jam Party」在上月（21日）再度展開，Ruby Taiwan 召集人鄧慕凡十分興奮地表示，非常地感動，過去因疫情停辦的聚會終於回來了，更感動的是，現場除了有資深的技術人，還有很多新面孔加入這個技術交流聚會，伴隨著現場令人垂涎的披薩美食、無限暢飲的冰涼飲料及汽水等軟性飲品，活動正式開始。

![2023 Ruby Jam Party 強勢回歸](https://i.imgur.com/tpuqTC1.jpg)

這次聚會剛好是 Ruby 30週年，在活動的一開始，主辦方即準備大蛋糕歡慶，現場更驚喜請到 Ruby 之父 Matz（松本行弘）透過影片方式跟在場技術人們打招呼，Matz 在短短十分鐘內，分享自己是如何在日本經濟泡沫下創造 Ruby 的心情，他表示，在日益茁壯的 Ruby 社群內，他感受到所有 Ruby 愛好者對 Ruby 的熱愛，尤其，每個人都願意投入自己、貢獻自己的力量，讓這個 Ruby 社群一直活絡，Matz 分享近況，他提到，現在花比較少時間在寫程式，但花更多時間在社群上做思考及管理，最後，他感性地透過影片對在場的 Ruby 愛好者告白，感謝所有人熱愛 Ruby 語言的人，讓他在工作上也因此變得更有意義！

![Ruby 之父 Matz 透過影片與大家打招呼，慶祝 Ruby30 週年](https://i.imgur.com/UQTNswq.png)


第一場 Ruby Jam Party 技術交流主題是「Ruby the command」由資深 Ruby 技術人大兜演繹 Ruby 如何與其他作業系統配合工作，以及如何輕鬆地替代 POSIX 實用工具、使用 Ruby 與其他 CLI 工具協作、透過Ruby構建 CLI 工具等，大兜在現場也不私藏的分享 Ruby 隱藏技能，十分有愛。

![資深 Ruby 技術人大兜分享](https://i.imgur.com/JzY5Vyj.jpg)

本次的 Ruby Jam Party 由 5xRuby（五倍紅寶石）及 PicCollage（拼貼趣）贊助，開放文化基金會協辦。5xRuby 的 Sho 表示很榮幸能跟 Ruby 愛好者們進行技術交流，因為 5xRuby 本身就是一間以 Ruby 語言為主的網路服務開發公司，並以做大型系統為主，非常貼近這個社群。 PicCollage 的 Grace 認為能齊聚一堂共享技術知識真的很棒，PicCollage 的 Alfred 則表示， PicCollage 公司內部也是以 Ruby 語言為主，很開心這個技術交流活動能延續下去，目前公司也有在徵求 Ruby 工程師，歡迎有興趣的工程師投履歷。
